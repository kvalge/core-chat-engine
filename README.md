# Core Chat Engine

A local AI chat application built with Spec-Driven Development (SDD) methodology.

## Goal

Build a chat engine that runs locally using open-source LLMs via Ollama,
with no external API costs. The project demonstrates AI-assisted development using
Spec-Driven Development principles.

## Development Tools

- **Spec Kit** — GitHub's open-source SDD toolkit. Used to generate constitution, 
  specification, technical plan and task breakdown before writing any code.
- **Kilo Code** — Open-source AI coding agent (VS Code extension). Used to implement 
  all tasks generated by Spec Kit, guided by the SDD workflow.
-**Cursor** - For fixing bugs and missing functionalities.

## Tech Stack

**Backend**
- Python 3.14 with full type annotations
- FastAPI — async REST API
- SQLite — local persistent storage
- Uvicorn — ASGI server

**Frontend**
- React 19
- TypeScript
- Vite

**LLM**
- Ollama — local LLM server
- llama3.2 — fast, lightweight model
- Mistral 7B — more capable model

## Features

- Multi-turn chat with SSE streaming (hand-rolled, no LLM SDKs)
- OpenAI-compatible API support (/v1/chat/completions)
- Tool use loop — date/time, web search, todo list, calendar - not implemented
- Projects with custom system prompts and per-project tool configuration
- File upload — text, PDF, images (vision-capable models only) - not implemented
- Backend configuration UI — add/edit/remove API backends
- API keys stored server-side

## Setup

**Requirements**
- Python 3.11+
- Node.js 18+
- Ollama installed and running

**Backend**
```bash
cd backend
pip install -r requirements.txt
uvicorn src.main:app --reload
```

**Frontend**
```bash
cd frontend
npm install
npm run dev
```

**Pull LLM models**
```bash
ollama pull llama3.2
ollama pull mistral
```

App runs at http://localhost:5173
API runs at http://localhost:8000
